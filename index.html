Engo

Engo is an open-source 2D game engine written in Go. It uses the Entity-Component-System paradigm. The code is available on GitHub. If you encounter problems, found bugs, or want to request a feature, you can do that here.

We're currently still developing this website at GitHub, so please be patient with us. Ideas and feature requests for the website may be filed here.

Donations will be used to pay for the domain registration.
